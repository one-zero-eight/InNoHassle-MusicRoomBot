# InNoHassle-MusicRoomBot

## Project Installation

1. Install dependencies with [poetry](https://python-poetry.org/docs/).
    ```bash
    poetry install
    ```
   
2. Setup environment variables in `.env` file.

    Run
    ```bash
    cp .env.example .env
    ```
   And edit .env


3. Run the bot
    ```bash
    python -m src.main
    ```
